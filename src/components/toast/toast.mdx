#### Toast is a component that displays brief, temporary notifications to users. 

*To use toast notifications, you must first add the `<Toaster />` component to your app's root.*

#### Configuration Options

The `<Toaster />` component accepts the following props:

- `position`: Determines where toasts appear ('top-left', 'top-right', 'bottom-left', 'bottom-right') [default: 'bottom-right']
- `duration`: How long toasts remain visible (in milliseconds) [default: 5000]
- `maxToasts`: Maximum number of toasts visible at once [default: 3]
- `usePortal`: Whether to render toasts in a React Portal [default: false]
